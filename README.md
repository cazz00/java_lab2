### Текст задачи:
Создайте сущность Дробь	со	следующими	особенностями:
• Имеет	числитель: целое число
• Имеет	знаменатель: целое число

3. Вывести на экран	примеры	и	результа

sub() - вычитание дробей.
multi() - умножение дробей.
div() - деление дробей.
minus() - вычитание целого числа из дроби.
